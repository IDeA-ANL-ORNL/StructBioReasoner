🧬 StructBioReasoner

AI-Driven Binder Design Workflow

1
Research Goal Generation
Input: Target protein name + sequence
Process: LLM generates research goal focused on probing interacting partners
Output: Research goal hypothesis
LLM Reasoning Goal Setting
Software: Jnana LLM Reasoner (currently OpenAI GPT-4o, migrating to vLLM/ALCF endpoints)
Hardware: CUDA GPU, Intel XPU
↓
2
Literature Mining for Interacting Partners
Process: Parse scientific literature to identify known interacting partners
Output: Hypothesis with target protein + interacting partner sequences
Literature Search Sequence Extraction
Software: HiPerRAG (High-Performance Retrieval-Augmented Generation)
Workflow Tools: Parsl / Globus Compute
Hardware: AMD GPU, Intel XPU, CUDA GPU
↓
3
Structure Prediction (Chai)
Process: Fold target sequence AND interacting partner sequences using Chai
Output: 3D structures of target and interacting partners
Chai Folding Structure Prediction
Software: Chai-1 (Structure Prediction)
Workflow Tools: Parsl / Globus Compute
Hardware: Intel XPU, CUDA GPU
↓
4
MD Simulations & Hotspot Identification
Process: Run MD simulations of target-partner complexes using MDAgent
Analysis: Cluster interaction sites to identify common binding hotspots on target
Output: Hotspot residues on target protein
MDAgent Clustering Hotspot Analysis
Software: OpenMM, MDAnalysis
Workflow Tools: Parsl / Globus Compute
Hardware: CPU, Intel XPU, CUDA GPU, AMD GPU (should work but not tested by our team)
↓
5
Peptide Hypothesis Generation
Input: Target protein + identified hotspots
Process: LLM generates hypotheses for starting peptide sequences likely to bind hotspots
Output: Proposed peptide sequences + target sequence
LLM Reasoning Peptide Design Hotspot Targeting
Software: Jnana LLM Reasoner (currently OpenAI GPT-4o, migrating to vLLM/ALCF endpoints)
Hardware: CUDA GPU, Intel XPU
↓
Strategy 6a: BindCraft Optimization
6a
BindCraft Design
Input: Proposed peptides + target sequence
Process: ProteinMPNN inverse folding + Chai forward folding + QC filtering
Output: Optimized peptides passing quality cutoffs
ProteinMPNN Chai QC Filters
Software: Chai-1, ProteinMPNN, MDAnalysis
Workflow Tools: Parsl / Globus Compute
Hardware: Intel XPU, CUDA GPU
Strategy 6b: Protein MLM
6b
Masked Language Model
Input: Proposed peptides
Process: Evaluate sequence uncertainty and propose mutations
Output: Mutated peptide variants
Protein MLM Mutation Proposal
Software: ESM (Evolutionary Scale Modeling) / GenSLM
Workflow Tools: Parsl / Globus Compute
Hardware: Intel XPU, CUDA GPU
↓
7
Decision: Continue Optimization?
Decision Maker: LLM Reasoner
Question: Are there enough high-quality passing peptides?
If NO: Loop back to Step 6 (BindCraft/MLM) with current best peptides
If YES: Proceed to MD simulations
Decision Point Quality Check
Software: Jnana LLM Reasoner (currently OpenAI GPT-4o, migrating to vLLM/ALCF endpoints)
Hardware: CUDA GPU, Intel XPU
🔄 OPTIMIZATION LOOP (Steps 6-7)
Iteratively refine peptides using BindCraft or Protein MLM until sufficient high-quality candidates are obtained
↓
8
MD Simulations of Peptide-Target Complexes
Input: Passing peptides + target protein
Process: Run MD simulations for each peptide-target pair
Output: Structures + trajectories for all complexes
MDAgent Complex Simulation Trajectory Analysis
Software: OpenMM
Workflow Tools: Parsl / Globus Compute
Hardware: Intel XPU, CUDA GPU, AMD GPU
↓
9
Binding Free Energy Calculations
Method: MM-PBSA (Molecular Mechanics Poisson-Boltzmann Surface Area)
Process: Calculate binding free energies for all peptide-target pairs
Output: Ranked peptides with binding free energies
MM-PBSA Free Energy Ranking
Software: Amber MMPBSA
Hardware: CPU
↓
10
Decision: Sufficient Top Peptides?
Decision Maker: LLM Reasoner
Input: Top-ranked peptides + binding free energies
Question: Are there enough high-affinity peptides?
If NO: Loop back to Step 5 using top peptides as new starting points
If YES: Proceed to final output
Decision Point Affinity Check
Software: Jnana LLM Reasoner (currently OpenAI GPT-4o, migrating to vLLM/ALCF endpoints)
Hardware: CUDA GPU, Intel XPU
🔄 REFINEMENT LOOP (Steps 5-10)
Use top-performing peptides as seeds for new design cycles until sufficient high-affinity binders are obtained
↓
11
✅ Final Output: Top Peptide Binders
Deliverables:
  • List of top-ranked peptide sequences
  • Binding free energies (ΔG)
  • Predicted structures of peptide-target complexes
  • MD trajectory data
  • Quality metrics and confidence scores
OUTPUT: Validated peptide binders ready for experimental validation
Software: Jnana LLM Reasoner (Report Generation)
Hardware: CUDA GPU, Intel XPU
💡 Tip: Hover over any step to see detailed software, hardware, and workflow tool information
