# System Requirements:
# - Python 3.8+ (required for async features and modern typing)
# - 8GB+ RAM recommended for large protein analysis
# - GPU recommended for deep learning models (CUDA-compatible)

# Installation Order:
# 1. Install conda/mamba for scientific packages
# 2. Install PyMOL via Homebrew: brew install pymol
# 3. Install OpenMM via conda: conda install -c conda-forge openmm
# 4. Install remaining packages via pip: pip install -r requirements.txt
# 5. Set OPENAI_API_KEY in .env file
# 6. Install Jnana separately from the Jnana repository

# Platform-Specific Notes:
# - macOS: Use Homebrew for PyMOL and system dependencies
# - Linux: Use conda-forge for scientific packages
# - Windows: Use conda/mamba for all scientific packages

# GPU Support:
# - CUDA 11.8+ recommended for PyTorch and OpenMM
# - Install PyTorch with CUDA: pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# Memory Requirements:
# - Minimum: 4GB RAM
# - Recommended: 16GB+ RAM for large protein systems
# - Large-scale simulations: 32GB+ RAM recommended
